Arrays and Array Methods

1. Array Basics
An array is an ordered list of values.
Each value has an index (starting from 0).

✔ Creating an Array
const fruits = ["Apple", "Banana", "Mango"];
const numbers = [10, 20, 30, 40];
const mixed = [1, "Hello", true, {a: 10}];
``

✔ Accessing Array Elements
console.log(fruits[0]);  // "Apple"
console.log(fruits[2]);  // "Mango"
``

✔ Updating Elements
fruits[1] = "Orange";
console.log(fruits);   // ["Apple", "Orange", "Mango"]

✔ Array Length
console.log(fruits.length); // 3

✔ Looping Through an Array
for loop:
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

for...of:
for (const fruit of fruits) {
  console.log(fruit);
}

2. Array Methods (Very Important Part)

A. Adding and Removing Elements
1. push() → Add at end
fruits.push("Grapes");
2. pop() → Remove from end
fruits.pop();
3. unshift() → Add at start
fruits.unshift("Papaya");
4. shift() → Remove from start
fruits.shift();

B. Searching & Checking
5. indexOf()
fruits.indexOf("Banana"); // returns index
6. includes()
fruits.includes("Mango"); // true

C. Getting Parts of Arrays
7. slice() → returns a portion (does NOT modify original)
const part = fruits.slice(1, 3);
8. splice() → remove/replace items (modifies original)
fruits.splice(1, 1);  // remove 1 item at index 1

D. Joining & Splitting
9. join()
fruits.join(", ");  // "Apple, Banana, Mango"
10. split() (works on strings)
"Apple-Banana-Mango".split("-");

E. Transforming Arrays (Most Important For Interviews)
11. map() → transforms each element and returns NEW array
const nums = [1, 2, 3];
const doubled = nums.map(n => n * 2);  
// [2, 4, 6]
12. filter() → keeps elements that return true
const result = nums.filter(n => n > 1);
// [2, 3]
13. reduce() → reduce array to single value
const sum = nums.reduce((acc, val) => acc + val, 0);
// 6
14. forEach() → loop but does NOT return new array
nums.forEach(n => console.log(n));

F. Other Useful Methods
15. find() → first matching element
const arr = [10, 20, 30];
arr.find(n => n > 15); // 20
16. findIndex()
arr.findIndex(n => n === 20); // 1
17. sort()
nums.sort();  // sorts as strings by default!
nums.sort((a, b) => a - b); // correct numeric sorting
18. reverse()
nums.reverse();


3. ES6+ Array Methods (Modern JavaScript)

19. Array.from()
Convert something to an array.
Array.from("ABC");   // ["A","B","C"]
20. Array.isArray()
Array.isArray(fruits); // true
21. flat()
[1, [2, [3]]].flat(2);   // [1, 2, 3]
22. flatMap()
[1, 2, 3].flatMap(n => [n, n*2]);
// [1,2, 2,4, 3,6]

1️⃣ What const actually means in JavaScript
const means:
“You cannot change the variable reference.”
It does NOT mean the value inside the array is frozen.
That’s why this is allowed:
const arr = [1, 2, 3];
arr.push(4);        // allowed
arr[0] = 100;       // allowed

But this is NOT allowed:
arr = [10, 20, 30];  // ❌ ERROR
Because you tried to reassign the entire variable.

2️⃣ What let means
let allows both:
✔ Mutation (changing array content)
✔ Reassignment (changing the whole array)
Example:
let arr = [1, 2, 3];
arr.push(4);            // allowed
arr = ["a", "b"];       // allowed
Both work fine.



Example Comparison
Using let
let nums = [1, 2, 3];
nums.push(4);         // OK → [1,2,3,4]
nums = [9, 9, 9];     // OK → reassigned
Using const
const nums = [1, 2, 3];
nums.push(4);         // OK → [1,2,3,4]
nums = [9, 9, 9];     // ❌ ERROR → Cannot reassign

🔥 Why can we mutate a const array?
Because const protects the variable binding, not the content.
const arr ------> [1,2,3]

arr cannot change where it points to,
but the array it points to can change.

✔ Use const (recommended 95% cases)
Because arrays are almost always mutated and not reassigned.
