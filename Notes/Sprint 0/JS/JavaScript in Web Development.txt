JavaScript in Web Development

⭐ 1) Working With Forms (Using JavaScript)
Forms are the main way to collect data from users.
JavaScript lets you:

Reading values .value, .checked, .selected
Read form input values
Validate inputs
Stop the form from reloading the page
Send form data to a server (using AJAX / Fetch)

Example Form
<form id="loginForm">
  <input type="text" id="username" placeholder="Enter username" />
  <input type="password" id="password" placeholder="Enter password" />
  <button type="submit">Login</button>
</form>

<p id="output"></p>

Handling form submission using JavaScript
const form = document.getElementById("loginForm");
const output = document.getElementById("output");

form.addEventListener("submit", function (event) {
  event.preventDefault(); // stop page reload

  const username = document.getElementById("username").value;
  const password = document.getElementById("password").value;

  output.textContent = `Username: ${username}, Password: ${password}`;
});

Explanation

event.preventDefault() → stops the browser from refreshing the page.
.value → gets the data typed by the user.
You can now validate or send this data to the server.

JavaScript

Examples of AJAX usage:

Logging in
Searching (auto‑suggest)

Getting products from a database
Loading content automatically

AJAX can be done using:
XMLHttpRequest (older way)
Fetch API (modern way)

⭐ 3) Fetch API (Modern AJAX)
Fetch allows you to make server requests asynchronously.

A) Basic GET Request
fetch("https://jsonplaceholder.typicode.com/posts/1")
  .then(response => response.json())
  .then(data => console.log("Data:", data))
  .catch(error => console.log("Error:", error));

Explanation

fetch(url) → sends a request.
.then(response.json()) → convert server response to JSON.
.then(data => ...) → access the actual data.
.catch(...) → handles network errors.

B) GET Request Using Async/Await (Cleaner)

async function loadPost() {
  try {
    const response = await fetch("https://jsonplaceholder.typicode.com/posts/1");
    const data = await response.json();
    console.log(data);
  } catch (err) {
    console.log("Error:", err);
  }
}

loadPost();


C) Sending Form Data using Fetch (POST request)
Let's use a form and send data to the server.
<form id="registerForm">
  <input id="name" type="text" placeholder="Name" />
  <input id="email" type="email" placeholder="Email" />
  <button type="submit">Register</button>
</form>

JavaScript
const form = document.getElementById("registerForm");

form.addEventListener("submit", async function (e) {
  e.preventDefault();

  const user = {
    name: document.getElementById("name").value,
    email: document.getElementById("email").value
  };

  try {
    const response = await fetch("https://jsonplaceholder.typicode.com/users", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify(user)
    });

    const result = await response.json();
    console.log("Saved:", result);

  } catch (error) {
    console.log("Error:", error);
  }
});

Explanation

method: "POST" → sending data.
headers → telling the server you're sending JSON.
body → the actual data (JSON.stringify(user)).
await response.json() → converting the server response to JSO
